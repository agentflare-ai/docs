## Environment Setup

### 1. Create Environment Variables

Create a `.env` file in your project root:

```bash
# Agentflare API credentials
AGENTFLARE_JWT_SECRET=your_jwt_secret_here
AGENTFLARE_PROJECT_ID=your_project_id_here

# Optional: Custom API endpoint
AGENTFLARE_API_URL=https://api.agentflare.com/v1

# Optional: Export configuration
AGENTFLARE_EXPORT_JAEGER=true
AGENTFLARE_JAEGER_ENDPOINT=http://localhost:14268/api/traces
```

### 2. TypeScript Configuration

If using TypeScript, add these compiler options:

```json tsconfig.json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "lib": ["ES2020"],
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true,
    "moduleResolution": "node"
  }
}
```

### 3. Python Configuration

For Python projects, create a configuration file:

```python config.py
import os
from agentflare_telepathy import Config

config = Config(
    api_key=os.environ.get("AGENTFLARE_JWT_SECRET"),
    project_id=os.environ.get("AGENTFLARE_PROJECT_ID"),
    # Optional settings
    batch_size=100,
    flush_interval=5.0,
    max_retries=3
)
```

## Verify Installation

### TypeScript/JavaScript

```typescript
import { TelepathyClient } from "@agentflare/telepathy-sdk";

// Verify installation
const client = new TelepathyClient({
  apiKey: process.env.AGENTFLARE_JWT_SECRET,
  projectId: process.env.AGENTFLARE_PROJECT_ID,
});

// Test connection
client
  .testConnection()
  .then(() => console.log("‚úÖ Telepathy connected successfully"))
  .catch((err) => console.error("‚ùå Connection failed:", err));
```

### Python

```python
from agentflare_telepathy import TelepathyClient
import asyncio

async def verify_installation():
    # Initialize client
    client = TelepathyClient(
        api_key=os.getenv("AGENTFLARE_JWT_SECRET"),
        project_id=os.getenv("AGENTFLARE_PROJECT_ID")
    )

    # Test connection
    try:
        await client.test_connection()
        print("‚úÖ Telepathy connected successfully")
    except Exception as e:
        print(f"‚ùå Connection failed: {e}")

# Run verification
asyncio.run(verify_installation())
```

## Common Installation Issues

<Warning>
  If you encounter installation issues, check our [troubleshooting
  guide](/docs/troubleshooting) or contact support.
</Warning>

### Permission Errors

If you get permission errors during installation:

```bash
# npm/yarn/pnpm
sudo npm install -g @agentflare/telepathy-sdk

# pip
pip install --user agentflare-telepathy
```

### Network Issues

Behind a corporate proxy? Configure npm/pip:

```bash
# npm proxy configuration
npm config set proxy http://proxy.company.com:8080
npm config set https-proxy http://proxy.company.com:8080

# pip proxy configuration
pip install --proxy http://proxy.company.com:8080 agentflare-telepathy
```

### Version Conflicts

Check for conflicting package versions:

```bash
# Check installed versions
npm list @agentflare/telepathy-sdk
pip show agentflare-telepathy

# Force specific version
npm install @agentflare/telepathy-sdk@1.2.3
pip install agentflare-telepathy==1.2.3
```
